To run file, use command prompt and navigate to the 'ncs-project' folder.
Run the command 'npm run dev' which will run webpack. The app should then be running on localhost:8080.

This project has all the basic requirements except that the interactions are not "disabled" even when the textfield is empty.

No optional requirements were implemented.

For ease of use, below is the npm command to install the needed packages:

npm i -S babel-loader babel-plugin-add-module-exports babel-plugin-react-html-attrs babel-plugin-transform-class-properties babel-plugin-transform-decorators-legacy babel-preset-es2015 babel-preset-react babel-preset-stage-0 react react-dom webpack webpack-dev-server
